The Operations Recommendations page provides a comprehensive list of operational improvement actions for your Azure environment.

## Features

- View all operations recommendations with KPI impact
- Filter by Observability Score to focus on monitoring improvements
- Group recommendations by Control to see the Potential Score Increase
- View additional KPIs: Backups, Availability, Active Resources
- Create tasks to track improvement work

:::tip
Group recommendations by "Control" to see which control groups will give you the highest Observability Score increase when fully completed.
:::
