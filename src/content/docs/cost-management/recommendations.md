The Cost Recommendations page surfaces Azure cost optimization opportunities, highlighting and prioritizing savings recommendations to bring clarity to cloud spend.

## Features

- View all cost-related recommendations with estimated savings
- Filter and sort by potential impact
- Drill into individual recommendation details with affected resources
- Track recommendation status and create tasks for follow-up

:::tip
Start by sorting recommendations by potential savings to identify the highest-impact optimizations first.
:::
