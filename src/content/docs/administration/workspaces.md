:::note[MSP / SA Feature]
Workspace management is available only for MSP (Managed Service Provider) and System Administrator users.
:::

Workspaces in vBox allow MSP users to organize and manage multiple customer environments from a single pane of glass.

## Key Features

- **Create Workspaces** — Set up dedicated workspaces for different customers or organizational units
- **Customer Management** — Add, edit, and manage customers within each workspace
- **Workspace Navigation** — Switch between workspaces from the sidebar
- **Settings Inheritance** — Configure default settings at the workspace level
