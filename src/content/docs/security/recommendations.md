The Security Recommendations page provides a comprehensive list of security improvement actions for your Azure environment, sourced from Microsoft Defender for Cloud and Microsoft Defender XDR.

## Features

- View all security recommendations with severity and compliance mapping
- Filter by Secure Score sub-category (Infrastructure, Apps, Identity, Data, Devices)
- See the **Potential Score Increase** for each recommendation
- Drill into recommendation details with affected resources
- Create tasks to track remediation work

:::tip
Filter recommendations by the "Secure Score: Infrastructure" scope to focus on the highest-impact security improvements that directly affect your Defender for Cloud Secure Score.
:::
