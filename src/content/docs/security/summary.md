The Security Summary page provides a high-level overview of your organization's security posture through embedded BI dashboards powered by Metabase.

## Dashboard Charts

The Security Summary dashboard includes the following visualizations:

- **Secure Score** — Overall security posture score
- **Secure Score: Infrastructure** — Infrastructure-specific security score trend
- **Security Benchmark** — Compliance benchmark tracking
- **Secure Score: Infrastructure Controls** — Breakdown by security control
- **Resource Health: Infrastructure** — Health status of monitored resources

:::note
The Security Summary page requires the [BI Analytics](/analytics/bi-analytics/) feature to be enabled.
:::
